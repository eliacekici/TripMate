**TripMate**

TripMate is a simple application that helps users plan and organize their trips easily.
It allows users to manage travel details in one place for a better travel experience.

**Prerequisites**

Before running this project, make sure you have the following installed:

Node.js
npm or yarn
React Native CLI
Android Studio (for Android emulator)
Git

**Installation**

1) Clone the repository:

git clone https://github.com/your-username/tripmate.git


2) Navigate into the project folder:

cd TripMateApp


3) Install dependencies:

npm install


**Running the Application**

1) Start the Metro bundler:

npx react-native start


2) In a new terminal, run the app on Android:

npx react-native run-android


The app will launch on the Android emulator or connected Android device.

**Optional (iOS – macOS only)**
npx react-native run-ios
